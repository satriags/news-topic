## Instalasi

Anda dapat menjalankan program pada repository "express-news-topic" ketik command berikut

```sh
npm run dev
```
Program akan berjalan pada server
```sh
http://localhost:6060
```
Program yang dibuat menggunakan NodeJS Framework Express dengan TypeScript.
